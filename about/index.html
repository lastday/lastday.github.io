项目经历

LYModelData（2016年2月-至今）

简介

iOS JSON 模型转换,支持相互转化，自主设计，独立开发第三方的库，采用runtime动态机制

总结

通过项目深入学习runtime机制，对其产生了一些自己的见解.
Runtime 学习笔记(总)记录自己的总结经验。

相对于其他JSON解析库优点：

  • 直接访问 ivar，则尽量使用 ivar 而不使用 Getter/Setter 这样节省一部分开销。
  • Key-Value Coding 使用起来非常方便，但性能上要差于直接调用 Getter/Setter，所以如果能避免 KVC 而用 Getter/Setter 代替，性能会有较大提升。

项目地址：LYModelData
